# Farkle Bot - Privacy Policy Repository

This repository hosts the official privacy and compliance documentation for the **Farkle Bot**, a community-created Discord bot developed exclusively for the **Kingdom Come: Deliverance (KCD) Discord server**.

## Purpose

The purpose of this repository is to provide a publicly accessible location for the Bot's legal and compliance documentation, including:

* Privacy Policy
* Terms of Service (if applicable)
* Future compliance-related updates and notices

## About the Bot

The Farkle Bot is a non-commercial project created for the enjoyment of the KCD Discord community. It provides an implementation of the classic Farkle dice game along with persistent player statistics and leaderboards.

The Bot is not intended for commercial distribution or use outside the KCD community.
